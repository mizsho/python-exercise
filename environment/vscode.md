# VSCodeのメモ

## マークダウン

### Markdown Preview Enhanced

- 使用方法
  - 参考：https://qiita.com/bounoki/items/bbad6de9f000fe1c283d
  - Markdown Preview Enhancedという拡張機能を入れる。
  - CSSをカスタマイズすると、少しかっこよくなる。

- 問題
  - プレビュー画面が勝手にスクロールされるバグがある。

### Markdown All in One

- 使用方法
    - プレビュー画面は規定のものを使用。(タブを右クリック=>プレビューを開く)
    - 項目番号、目次生成機能がある。

- 問題
  - CSSのいじり方はわからない
